Color themes

This template was thought with 5 color themes in mind but you can add your own easly using the LESS file.

The LESS file contains at the begining the default colors used but also in themeDefs mixins there are the other color themes.

For a smaller css file I suggest deleteing the themeDefs not used and change @numberOfThemes variable to the number of themes you want.

For example you like the blue color theme so remove themeDefs 1,2,4 and 5 and rename the themeDefs(3) to themeDefs(1) and compile the LESS file again.

.themeDefs(1) {
@dark-color: #7946bd;
@light-color: #927CBF;
@danger-color: #D94A56;
@success-color: #CF89B1;
@info-color: #BF99AE;
@warning-color: #F2DA91;
@background-color: #F6F6F6;
@slider-overlay-color: #927CBF;
}

If you don't want to recompile the LESS file you can change the class on body with .theme-3 class.

Change the slider images

To change the slider images replace the background-image: url() from .top-slider items, with the link to your image

Disable animations

To disable the animations, remove from main.js file the function initElementTransitions.

Static image banner

To disable the slider and use only on image as a static banner you need to add to body the class .no-slider.

Keep only one item in .top-slider and remove the links for navigation that have the class .carousel-control

Change the portfolio filters

Every item in portfolio has a class for example .apps which is the same as the the data-filter attribute from the buttons in .portfolio-filters.

An items can have more classes that match multiple filters.

Change the portfolio images

Every item from portfolio uses data-image attribute for loading the image on the main screen.

Change the team images

Every team-member uses a img for loading the images. Edit this element to load the desired image.

Change marker position on the map

The map center can be modified from main.js file in initMap() function.

For centering the map modify address under map.

For chaning the map marker modify address under marker.

Contact form - making it work

Contact form is sent using an Ajax call to contact.php file.

Modify $to = 'Your Name '; with your email address.

Use icons

This theme uses the Entypo icon pack.

In the file demo-icons.html you will find all the icons available to use and the class for them.

Grunt

The grunt file has included a less compile task that also minifies and generates a source map and a JS minifier.

I suggest to use them for production sites.
